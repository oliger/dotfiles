# Dotfiles

## Requirements

- [XCode](developer.apple.com/downloads)
- [Jetbrains Mono](https://www.jetbrains.com/lp/mono)

## Installation

```sh
git clone git@github.com:oliger/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
scripts/bootstrap
```
